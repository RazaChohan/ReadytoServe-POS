###Ready2Serve POS###
“Ready2Serve POS” is a console based application for a fast food restaurant’s drive through  

* A user; in this case a sales man (main stakeholder of the system) is able to place the order, view the order (while adding items/deals or after adding them) 

* He/she can also make changes in items/deals added, quantity while reviewing the order

* At once the items/deals are added in the order and an invoice is generated, any changes cannot be made in that order

* Another user of the system is administrator who can view all the sales, items, deals and can manage them.
